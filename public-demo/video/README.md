# Video Runner

This folder contains a lightweight HTML presentation that can be used like a simple demo video.

It is meant to support the public story behind this project:

- you understood the SharePoint process well
- you knew the outcome you wanted
- you used Codex in VS Code to help bridge the technical gap
- you are now sharing a free starter resource so others can try a similar approach

## File

- `index.html`

## What it does

- plays the story as a timed slide sequence
- shows the before and after mockups
- includes voiceover text for each scene
- can autoplay or be stepped through manually

## How to use it

1. Open `index.html` in a browser.
2. Put the browser in full screen.
3. Record the screen with QuickTime, Loom, Teams, Zoom, or any screen recorder.
4. Read the voiceover on the right panel or use it as subtitle guidance.

## Controls

- `Space`: play or pause
- `Right arrow`: next scene
- `Left arrow`: previous scene

## Why this approach

There is no direct video-render pipeline configured in this project right now, so this is the fastest public-safe way to create a polished walkthrough that you can turn into a real MP4 in a few minutes.
