# Public Demo Pack

This folder contains a public-safe version of the planning workspace story so it can be shared externally without exposing real people, departments, sites, or operational language.

It is also designed as a free starter resource for people who want to explore a similar workflow in VS Code with Codex.

## What this pack includes

- `LIST_TEMPLATE.md`: Generic SharePoint / Microsoft Lists structure
- `sample-team.csv`: Placeholder assignees in English
- `sample-tasks.csv`: Placeholder activities in English
- `images/before-sharepoint-list.png`: Mockup of a basic SharePoint list view
- `images/after-planning-workspace.png`: Mockup of the current planning app experience
- `images/before-after-comparison.png`: Side-by-side comparison graphic for LinkedIn
- `STORY_SCRIPT.md`: Public-safe script for LinkedIn, video, or presentation
- `CODEX_STARTER_GUIDE.md`: How others can apply the same pattern with Codex in VS Code
- `DISTRIBUTION_PLAN.md`: How to publish the post and share the free resource
- `video/index.html`: Browser-based video runner for recording a walkthrough
- `LINKEDIN_POST.md`: Draft post for LinkedIn
- `PUBLIC_REPO_CHECKLIST.md`: Checklist before publishing anything publicly

## Public naming used here

- Product name: `Team Planning Workspace`
- List name: `Team Planning Data`
- People: `Worker 1` to `Worker 8`
- Activities: `Activity 1` to `Activity 12`
- Activity types: `Activity Type 1` to `Activity Type 6`

## Share safely

This folder is safe to use as a showcase starter pack.

The project root is not fully sanitized yet. If you want to publish the full repository, first review `PUBLIC_REPO_CHECKLIST.md`.

## Recommended public use

- Use the images and video runner for the LinkedIn post
- Use `LINKEDIN_POST.md` and `STORY_SCRIPT.md` for your message
- Share this folder, or a sanitized repo built from it, as the free starter resource
